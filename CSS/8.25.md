- link和@import的区别
link 是XHTML标签，除了加载 CSS之外，还可以定义RSS等其他事务；@import属于css范畴，只能加载css
link 是XHTML标签，无兼容问题；@import属于css2.1提出，低版本浏览器不支持
link 支持使用javascript控制dom去改变样式；@import不支持
link 引用css时，在载入页面时同时加载，@import则需要页面完全载入后加载
